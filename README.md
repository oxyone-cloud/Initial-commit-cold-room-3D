# Initial-commit-cold-room-3D
Interactive 3D Cold Room Simulator (Model CR-274). Built with Java (Spark) and Three.js, this app features a realistic industrial dashboard and an exploded 3D view of a 2.74m x 1.83m x 2.00m unit. It simulates assembly steps and live temperature monitoring (-18.4°C) with a modern, clean UI. Ideal for technical training and IoT visualization.
